== Description ==

Add Dummy Post plugin allows user to add post in their WordPress website.

= FEATURES =

- It allows users to add the dummy post for the multiple use like check the UI/UX of the post listing and inner page.

= STEPS TO USE =

- To use this plugin, you just need to go the Add Dummy Post page under the Tools tab.
- Then you just need to add the number of post there to add in your WordPress website.
